The official RealTimeCrowd.net plugin for WordPress and WooCommerce.

== Description ==

This plugin enables Real-Time Crowd tracking and generation on your WordPress site. See what others are looking at, right now.
Website: [www.RealTimeCrowd.net](http://www.RealTimeCrowd.net/)

== Installation ==

1. Download and extract the zip archive
2. Upload the 'real-time-crowd' directory into the '/wp-content/plugins/' directory on your server.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Using the "WordPress *General* Settings" menu, navigate to Real-Time Crowd and enter your RealTimeCrowd.net Account alias.
5. Add the widget to a sidebar.
